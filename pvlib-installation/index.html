    Installation

    Für die Installation der PvLib-pv3 wird git benötigt. Sollte git noch nicht installiert sein, muss dies zuerst getan werden.

    Git installieren

    Git Repository clonen

    Im ersten Schritt wird das git repository lokal geclont. Dafür wird zunächst zu dem Ordner navigiert, wo das Projekt geclont werden soll. Anschließend wird das git Terminal geöffnet (Rechtsklick -> git bash here) und der folgenden Befehl eingegeben.

    • git clone https://github.com/htw-pv3/pvlib.git

    Nun wird das Projekt lokal erstellt.

    Installieren der benötigten Module

    Innerhalb des Codes werden Bibliotheken verwendet. Damit dies funktionieren kann, müssen die benötigten Module installiert werden.

    Virtuelle Umgebung erstellen

    In Python wird für jedes Projekt dafür sogennante virtual environments (kurz "venv") verwendet. Es gibt mehrere Möglichkeiten diese zu erstellen. Eine Möglichkeit ist die Verwendung von der in pip integrierten venv Funtkion.

    Zuerst wird das Terminal geöffnet und zu dem Projekt navigert. Dies kann mit dem Befehl cd <Pfad> erledigt werden. Mit dem folgenden Befehl wird dann eine virtuelle Umgebung mit dem Namen "env" erstellt. Dadurch werden in dem aktuellen Verzeichnis die Ordnerstruktur der venv erstellt.

    • python3 -m venv env

    Installieren der Module in der erstellten Umgebung

    Nun muss zunächst im Terminal die neu erstellte Umgebung aktiviert werden. Dabei ist darauf zu achten, dass der richtige Name der Umgebung genommen wird. Wenn der vorherige Befehl genau so verwendet wurde, lautet der Name der Umgebung "env" und die folgenden Befehle können verwendet werden.

    • Linux and Mac: source env/bin/activate
    • Windows: env/Scripts/activate.bat

    Wenn die Aktivierung erfolgreich war, erscheint im Terminal in der Befehlszeile der Name der Umgebung in Klammern. Im letzten Schritt werden dann die Module installiert:

    • pip install -r requirements.txt

    Der Installationsvorgang ist abgeschlossen, sobald im Terminal wieder die Befehlszeile erscheint. Durch den Befehl deactivate kann die venv wieder deaktiviert werden.
